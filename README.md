# DunksJams

A Unity 6000 (URP) game project with a toolkit vibe: a stash of reusable systems for fast iteration, clean gameplay
plumbing, and editor-friendly workflows.

## What it includes

- Event system for decoupled messaging (with pooled events).
- Singleton-based manager pattern with auto-create support.
- Finite state machine framework for state-driven logic.
- AI utilities: steering behaviors and flow-field pathfinding.
- Tweening and easing helpers, including curve generation and previews.
- Math, DSP, and noise utilities for procedural work.
- Custom data structures: ring buffers, queues, LRU cache, spatial hashes, pools.
- Extension methods for common Unity types.
- Editor utilities: graph view tooling, foldout scopes, persistent editor values, kernel generator.
- Edit Mode tests for core utilities, data structures, extensions, and tweening.
